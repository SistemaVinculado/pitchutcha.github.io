Escala Tipográfica (Dentro de um Artigo)

Este é um Título H1

Este é um parágrafo padrão. A altura de linha (`1.65`) torna o texto longo mais fácil e confortável de ler. A cor do texto principal é `var(--text-primary)` e a cor secundária é `var(--text-secondary)`.

Este é um Título H2

Um link dentro do texto se parece com isto, usando a cor primária do site. Texto em negrito usa a cor de texto primária para maior destaque.

Este é um Título H3

Listas também seguem o padrão de altura de linha:

  • Item de lista um.
  • Item de lista dois.

Este é um Título H4

Esta é uma citação (blockquote). Ela é usada para destacar informações importantes ou citações, com uma borda lateral e estilo itálico para diferenciação visual.
