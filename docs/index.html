Paradigmas Essenciais

Explore diferentes abordagens para resolver problemas e veja como elas funcionam na prática com exemplos visuais.

Dividir para Conquistar+

Quebre um problema grande em subproblemas menores, resolva-os independentemente e combine as soluções. O Merge Sort é um exemplo clássico desta técnica poderosa.

Programação Dinâmica+

Evite recálculos armazenando os resultados de subproblemas (memoização). Ideal para problemas de otimização, como o cálculo da sequência de Fibonacci.

Algoritmos Gulosos (Greedy)+

Faça a escolha localmente ótima em cada etapa, na esperança de encontrar uma solução globalmente ótima. Eficiente para problemas como o da seleção de atividades.

Busca Binária
Um algoritmo de busca eficiente que encontra a posição de um item em uma lista ordenada. Reduz pela metade o tamanho da busca a cada iteração.
schedule Complexidade: O(log n) 

# Exemplo de Busca Binária em Python

def binary_search(arr, target):
    low = 0
    high = len(arr) - 1

    while low <= high:
        mid = (low + high) // 2
        guess = arr[mid]

        if guess == target:
            return mid  # Encontrado!
        elif guess > target:
            high = mid - 1
        else:
            low = mid + 1
            
    return None # Não encontrado

# Teste:
my_list = [1, 3, 5, 7, 9, 11]
print(binary_search(my_list, 7))


// Exemplo de Busca Binária em JavaScript

function binarySearch(arr, target) {
    let low = 0;
    let high = arr.length - 1;

    while (low <= high) {
        const mid = Math.floor((low + high) / 2);
        const guess = arr[mid];

        if (guess === target) {
            return mid; // Encontrado!
        } else if (guess > target) {
            high = mid - 1;
        } else {
            low = mid + 1;
        }
    }
    
    return null; // Não encontrado
}

// Teste:
const myList = [1, 3, 5, 7, 9, 11];
console.log(binarySearch(myList, 7));
